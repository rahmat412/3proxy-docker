# safe-proxy

A lightweight 3proxy Docker image for deployment on Render.com.

## Ports
- `3128` — HTTP proxy
- `1080` — SOCKS5 proxy

## Deploy on Render.com
1. Fork this repo
2. Go to render.com → New → Web Service
3. Connect this repo
4. Set Runtime to **Docker**
5. Set Region to **Singapore**
6. Deploy
